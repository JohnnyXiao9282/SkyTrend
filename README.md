# Weather Trend Forecasting Project

## Overview
This project analyzes the "Global Weather Repository.csv" dataset to forecast future weather trends and demonstrate data science skills through both basic and advanced techniques.

## Project Structure
- `data/`: Raw dataset
- `notebooks/`: Jupyter notebooks for EDA and modeling
- `src/`: Python scripts (if needed)
- `outputs/`: Generated figures, models, and results
- `report/`: Final report or presentation

## How to Run
1. Install dependencies from `requirements.txt`
2. Open the main notebook in `notebooks/`
3. Follow the steps for data cleaning, EDA, modeling, and analysis
